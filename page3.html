The Instructions to make your own

This page has step-by-step instructions on how to use HTML on Notepad

INSTRUCTIONS:

STEP 1: Open a new notepad document
STEP 2: Type HTML basic code in the document
STEP 3: Select a file in the notepad menu at your top right
STEP 4 Then, select "save as" to save the file
STEP 5: Select the file name as anything you want plus, ".html" and make sure it has UTF8
STEP 6: Select the file type as "all files", this step is important.
STEP 7: Open the file in a browser by double-clicking on the file.
